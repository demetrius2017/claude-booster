## Problems / Solutions

| Problem | Solution |
|---------|----------|
| `tr` subprocess on hot path (~200 calls/session) | Replaced with `read -r` + `${content//[[:space:]]/}` (bash builtins) |
| `read -r` returns exit 1 on EOF without newline | Use `;` not `\|\|` — `read` populates variable regardless of exit code |
| model_tag_enforcer blocks Agent calls for its own fix | Used `codex_sandbox_worker.sh` (returns diff), applied via Edit |
| delegate_gate exhausts 1-edit budget mid-task | TaskCreate resets budget; learned to plan edits around budget |
| `_infer_category()` missing categories | consilium/audit/trivial never matched → all routed as "medium" |
| `_HIGH_BLAST_KEYWORDS` diverged from model_balancer.py | Expanded to superset covering both naming conventions |
| `${CLAUDE_HOME}` in statusLine template = hardcoded after install | Switched to `~/.claude/` (shell expands `~` natively) |
| statusLine not visible in current session | Settings read at session start; requires restart to take effect |
