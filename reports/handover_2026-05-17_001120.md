## What was done

### /go command (`8f054a8`)
Hardcoded тройка pipeline as a slash command. Lead invokes `/go` after formulating an Artifact Contract. The command mechanically executes:
- Phase 0: AC validation (5 mandatory fields)
- Phase 1: Flow Designer (foreground, Opus) → produces PFD
- Phase 2: Worker + Verifier (parallel, background, Sonnet)
- Phase 3: Run Verifier's test script
- Phase 4: Verdict (PASS/FAIL with W/V/A/E classification on failure)

No skip path inside /go. If the task is trivial enough to skip Flow Designer, don't use /go.

### go_gate.py (`88ac3fa`, `58b83e7`)
PreToolUse hook that watches Agent spawns. If an Agent is spawned with a coding description (contains "worker", "implement", "fix", "refactor", etc.) and no `.go_active` marker exists, the hook blocks with exit 2. This enforces /go as the only path for paired delegations.

Shortened stderr to 14 chars to avoid noise. Added description-prefix detection for agents spawned by /go itself.

### delegate_gate SSH fix (`6e180ff`)
Root cause: `_segment_is_recon()` had guards in wrong order. The `$()` command substitution guard (generic, for local commands) fired before SSH-specific logic. SSH heredocs commonly contain `$(...)` which is legitimate remote execution.

Fix: moved SSH detection before the `$()` guard as an early-exit. If the command is SSH and contains no destructive patterns → immediately return True (recon). 32 test assertions confirm correctness.
