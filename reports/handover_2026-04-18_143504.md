## Problems / solutions

| Problem | Solution |
|---------|----------|
| Cygwin/MSYS/MinGW bypassed the `platform.system() == "Windows"` refusal | `startswith(("CYGWIN","MSYS","MINGW"))` check at `install.py:196` |
| WSL2 + Claude Code Desktop-on-Windows = silent misinstall (different HOMEs) | `_detect_wsl()` warns at preflight about the side-where-Claude-runs requirement |
| WSL1 drvfs SQLite WAL corruption on first DB write | `_detect_wsl() == "wsl1"` → refuse with exit 13 |
| Snap/Flatpak-sandboxed Claude Code can't read `~/.claude/` (AppArmor) | `$SNAP` / `$FLATPAK_ID` env check → refuse with exit 14 |
| NFS/CIFS/SMB home: SQLite WAL forbidden per SQLite docs | `/proc/mounts` longest-prefix match + 9 network-fs types → refuse exit 15 |
| Backup on same volume as CLAUDE_HOME (external drive unmount = unrecoverable) | Stage in `$TMPDIR`, finalize to `~/.claude/backups/` only after full success |
| `${PYTHON}` = `sys.executable` baked absolute path broke on `brew upgrade python` | `_resolve_python()` prefers `shutil.which("python3")` (stable symlink) |
| `permissions.ask`/`deny` lists silently dropped in merge | Loop over all three keys, union each into sorted set |
| `os.fsync` on Darwin ≠ platter flush | Darwin-gated `fcntl.F_FULLFSYNC` in `atomic_write` |
| Apple Python 3.9 may lack FTS5 → runtime fail, not install-time | Preflight probe: `CREATE VIRTUAL TABLE USING fts5(...)` → exit 16 |
| Idempotency broken after `{{GIT_AUTHOR_NAME}}` substitution was added (raw template sha ≠ post-substitution installed sha) | `_effective_src_sha()` hashes the post-substitution bytes — second run = 0 writes |
| `reports/` folder had 44 `/Users/dmitrijnazarov/...` paths + project names + IBKR operational intelligence | Scrubbed `add_frontmatter.py` / `index_reports.py` / `rolling_memory.py` in templates; `git rm --cached -r reports/` (kept locally); pushed from fresh `/tmp/claude-booster-public` init with no history baggage |
| `README.md` felt technical, not compelling | Rewrote opening as session-level pain narrative; added before/after table (11 scenarios) + pain→fix map (9 rows) + "How it actually works" section |
| Audit report self-matched its own secret-scan regex | Concatenate string literals in pattern definitions so the doc does not trip its own scanner (`"v" + "less://"`) |
